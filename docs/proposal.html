Inertia

We propose to create a futuristic racing game reminiscent of F-Zero for SNES (1). We plan to utilize the concepts learned in CS184 to create a modern game with similar gameplay, akin to WipEout HD for PS3 (2). The design challenge is to allow the user to interface to play the game, rendering the graphics with modern techniques, and maintaining an acceptable framerate on typical computers. We plan to extend our classical rendering pipeline from AS7/AS8 to include arbitrary movement with an independent velocity vector and friction coefficients of a racing ship on a b-spline track. The ship will allow for quaternion rotations based on a function of velocity, acceleration, and a friction coefficient. Furthermore, we will build upon general rendering techniques by adding soft shadows(3)(4), HUD rendering, and light scattering (5)(6). For these topics, we will investigate multiple methods to attain acceptable performance. If time permits, we may pursue a motion blur effect (7), or add multi-player functionality.
